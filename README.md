# Focus Tea - Versão Premium

Estrutura com landing page, login/cadastro, dashboard e páginas separadas.

## Arquivos
- index.html: landing page pública
- login.html / cadastro.html: entrada do usuário
- dashboard.html: painel principal
- paginas/: rotina, estudos, exercícios e configurações
- css/style.css: visual completo
- js/app.js: funções com localStorage
