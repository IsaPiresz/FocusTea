# Focus Tea

Protótipo para o Desafio Liga Jovem.

## Estrutura

- `index.html`: página pública de apresentação
- `login.html`: entrada simulada
- `cadastro.html`: cadastro simulado
- `dashboard.html`: painel do usuário
- `paginas/rotina.html`: rotina funcional
- `paginas/estudos.html`: estudos com IA simulada
- `paginas/exercicios.html`: exercícios
- `paginas/configuracoes.html`: acessibilidade e temas
- `css/style.css`: design do site
- `js/app.js`: funções e localStorage
